## Organização de pastas do projeto: 

- lib:
  - *models*: Definição das classes relacionadas as entidades do projeto
  - *widgets*: Componentes reutilizáveis
  - *rotas*: Definição das rotas de navegação entre as telas
  - *pages*: Telas do app
  - *data*: Camada de acesso ao banco de dados
  - *utils*: Implementações que podem ser reutilizadas no app
  - *controller*: Camada intermediária que irá receber, e indicar as respostas dos processamentos.
